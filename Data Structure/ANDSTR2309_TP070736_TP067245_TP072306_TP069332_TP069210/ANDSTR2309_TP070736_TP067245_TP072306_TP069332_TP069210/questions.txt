What is a bloom filter?/A probabilistic data structure used to test whether an element is a member of a set, which may return false positives but not false negatives./6
What is a cuckoo hashing?/A hashing technique where each key can be stored in one of two possible locations, and when a collision occurs, the key is 'kicked out' to its alternate location./7
What is a perfect hash function?/A hashing function that generates a unique hash value for each possible input, eliminating collisions in the hash table./8

What is the chromatic number?/The smallest number of colors needed to color the vertices of a graph such that no two adjacent vertices have the same color./6

What is a greedy algorithm?/An algorithm that follows the problem-solving heuristic of making the locally optimal choice at each stage with the hope of finding a global optimum./6
What is a divide and conquer algorithm?/An algorithm that breaks a problem into smaller subproblems, solves the subproblems recursively, and then combines their solutions to solve the original problem./7
What is a backtracking algorithm?/An algorithmic technique for solving problems by systematically searching through all possible configurations until a solution is found or all possibilities have been explored./8
What is a branch and bound algorithm?/An algorithmic technique for solving optimization problems by partitioning the search space into smaller subspaces and bounding the search within those subspaces./9
What is a brute force algorithm?/A straightforward approach to problem-solving that exhaustively tries every possible solution until a satisfactory one is found./5
What is a heuristic algorithm?/A problem-solving technique that employs a practical method, not necessarily optimal or perfect, to find a solution to a problem./6
